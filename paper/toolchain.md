# Toolchain / ツールチェーン

## English

The implementation artifact is `presence-audit`.

It includes:

- JSON Schema for PRESENCE YAML specifications.
- CLI audit command.
- Markdown and JSON report generation.
- Example system specifications.
- GitHub Actions workflow example.

The tool can be used as:

```text
python presence-audit/cli/presence_audit.py audit presence-audit/examples/cloud_wellbeing_dashboard.yaml
```
