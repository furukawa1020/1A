# Embedded Device Gateway Template

## English

This template shows an embedded-style gateway that uses the C ABI sketch. It only allows C2 self-only cues and falls back to a neutral state when a claim is denied or rewritten.

The demo source is copied from the same API shape as `presence-ffi-c/include/presence_guard.h`.

## 日本語

このtemplateは、C ABI sketchを使うembedded-style gatewayを示す。C2のself-only cueだけを許可し、claimがdenyまたはrewriteされた場合は中立表示に戻す。
