## Use Limits / 利用上の制限

These sources support reference-profile construction only. We do not use logged-in screens, private documents, scraping, API probing, vulnerability testing, or claims about actual deployments.

これらのsourceはreference profile構築だけに用いる。ログイン後画面、非公開資料、scraping、API探索、脆弱性診断、実運用に対する断定には用いない。
