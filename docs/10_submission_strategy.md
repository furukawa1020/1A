# Submission Strategy

## Primary Target

USEC 2027-equivalent is the first target because the study is a usable security/privacy contribution centered on human factors, privacy risk, and system design.

## Secondary Target

SOUPS 2027-equivalent is a stronger venue, but Paper 1A alone may be thin. A later submission should ideally include the follow-up interactive system study.

## Domestic / Journal Backup

JIP Human-Centered Security, Privacy and Trust is a plausible backup target for a peer-reviewed domestic or journal path.

## Avoid For Paper 1A Alone

ACM CCS and USENIX Security main conference are too risky for Paper 1A alone unless the work is expanded with a stronger system artifact, real-world evidence, preregistered analysis, and broader security framing.
