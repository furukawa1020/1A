# Measure Definitions

All participant-rated items use a 7-point agreement scale:

- 1: まったくそう思わない
- 7: 非常にそう思う

## MFS: Monitoring Feeling Score

Monitoring feeling is measured with five items:

- 見張られている感じ
- 管理されている感じ
- 本人支援より評価・管理に近い感じ
- 不利に使われそうな感じ
- 不調を隠しにくくなる感じ

## LIS: Label Imposition Score

Label imposition is the core measurement of imposed interpretation:

- 状態を決めつけられた感じ
- システム判断が自分の感じ方より優先される感じ
- 納得していない意味づけ
- 解釈が狭められる感じ
- 自分で考える余地の減少

## SOUS: Self-Observation Utility Score

SOUS is not general usability. It measures whether the condition supports self-observation:

- 見直すきっかけ
- 無理して続けているかもしれないと考える余地
- 判断を急がされず考えられること
- 自分の文脈で解釈できること
- 不安を過度に増やさず振り返れそうなこと

## WU: Willingness to Use

WU measures acceptance of use in daily work/study contexts.

## WD: Willingness to Disclose

WD measures whether the participant would allow the system to handle state-related information.

## Free Text

Every condition includes:

> この条件を「支援」と感じたか、「監視」と感じたか。その理由を書いてください。

Participants are instructed not to write identifying information or concrete disease/diagnosis names.
